# CSS Loader for Millennium v1.1

Version 1.1 is a reliability release built from feedback during the first 1.0
testing cycle.

## Fixes and improvements

- Fixed downloaded profiles containing emoji or other UTF-8 characters being
  rejected by the Windows backend.
- Fixed image-picker assets bundled inside profiles not being restored to their
  dependency themes.
- Added traversal protection when restoring profile assets.
- Fixed dependency installation failures being ignored.
- Fixed the Store reporting a successful installation when the backend had
  returned an error; the actual failure is now shown.
- Fixed packaged backends that could crash because Pillow's native `_imaging`
  component was missing.
- Backend builds now clear PyInstaller analysis caches before packaging.
- Added a single-instance desktop guard so repeated launches focus the existing
  window instead of opening duplicates.
- Added regression coverage for Unicode profiles, profile asset restoration,
  and unsafe asset paths.

## Included artifacts

- Windows x64 MSI installer
- CSS Loader for Millennium backend executable
- Bundled CSS Loader Companion for Millennium 1.1
