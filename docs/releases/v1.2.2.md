# CSS Loader for Millennium v1.2.2

Version 1.2.2 restores stock CSS Loader target matching across Millennium's
direct-injection runtime.

## Changes

- Publishes resolved injections with explicit `title`, `url`, or `class` target
  types and their original match values.
- Anchors regular-expression title targets against the full document title.
- Preserves `~value~` targets as page-URL substring matches and `!value` targets
  as HTML class matches.
- Retains the protocol-v1 `matchRegex` field so the backend and companion can
  update independently without breaking existing title-targeted styling.
- Uses stable, collision-resistant identifiers for custom targets.
- Adds compiler tests for title, URL, and class serialization.

Install CSS Loader Companion v1.2.1 separately for full Store, Community, and
other isolated BrowserView support. Themes, profiles, settings, and the selected
Millennium theme remain unchanged.
