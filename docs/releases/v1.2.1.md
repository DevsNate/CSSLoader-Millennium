# CSS Loader for Millennium v1.2.1

Version 1.2.1 moves the direct-injection runtime mailbox into the separately
installed CSS Loader Companion plugin.

## Changes

- Publishes `build-report.json` and `runtime-state.json` to
  `%STEAM%\millennium\plugins\css-loader-companion\runtime`.
- Stops generating a metadata-only Millennium theme and `skin.json`.
- Removes the obsolete `%STEAM%\millennium\themes\CSS Loader` mailbox after a
  successful publish.
- Refuses to recreate a partial companion directory when the plugin is not
  installed.

Install CSS Loader Companion 1.2.0 separately before opening the desktop app.
Themes, profiles, settings, and the selected Millennium theme remain unchanged.
