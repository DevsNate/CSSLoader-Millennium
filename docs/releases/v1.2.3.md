# CSS Loader for Millennium v1.2.3

Version 1.2.3 adds mode-aware DeckThemes catalog scopes so Desktop and Big
Picture themes stay in their intended Steam interface.

## Changes

- Fixes Desktop Store themes such as Prominent Store Prices affecting Big
  Picture Store pages.
- Caches DeckThemes catalog targets during theme installation.
- Backfills catalog metadata for existing UUID-based themes.
- Publishes `desktop`, `gamepad`, or `all` scope with each runtime injection.
- Preserves stock CSS Loader behavior when catalog metadata is unavailable.
- Supports `.css-loader-scope.json` as an override for local themes.
- Prevents local scope metadata from being included in uploaded theme ZIPs.
- Adds regression coverage for catalog classification, caching, overrides,
  backfill, and runtime serialization.

Install CSS Loader Companion v1.2.2 separately to enforce these scopes across
Steam's direct documents and isolated Store BrowserViews.
