# Attribution and provenance

This repository combines and adapts work from the CSS Loader ecosystem for a
Millennium-first Windows runtime.

- The Python runtime is derived from
  [SDH-CSSLoader](https://github.com/suchmememanyskill/SDH-CssLoader) and the
  DeckThemes CSS Loader project.
- The desktop manager is derived from
  [DeckThemes/CSSLoader-Desktop](https://github.com/DeckThemes/CSSLoader-Desktop).
- Millennium integration uses the public plugin APIs and build tooling from
  [SteamClientHomebrew/Millennium](https://github.com/SteamClientHomebrew/Millennium).

Those projects retain their respective copyright notices and attribution.
This combined distribution is provided under GPL-3.0-only; see `LICENSE`.
Installed themes are separate works and remain under the terms chosen by their
authors. They are not included in this repository.
